# Configuration Reference

The config file lives at `docs/monitoring/config.yml` in each monitored project. It is generated by `/forge-init` and can be edited manually.

## Schema Version

```yaml
version: 1  # required, must be 1
```

## Project

```yaml
project:
  name: my-project              # display name
  github_repo: owner/repo       # GitHub slug
  base_branch: main             # target for PRs
  mode: observe                 # observe | fix
  workspaces:                   # for running tests on fixes
    - name: app
      path: .
      test_command: "npm test"
      build_command: "npm run build"
      typecheck_command: "npm run typecheck"
```

## Servers (informational)

```yaml
servers:
  - name: production
    display_host: 10.0.0.1      # shown in reports
    display_user: root           # shown in reports
```

Connection details (SSH keys, host, user) live in GitHub repo secrets, not here.

## Log Bridge

```yaml
log_bridge:
  type: github-actions-branch
  branch: forge-logs             # written by Level 0 collector
  logs_path: logs                # directory within the branch
  max_age_hours: 3               # warn if branch is stale
```

## Collector (Level 0)

```yaml
collector:
  type: github-actions
  workflow_file: .github/workflows/forge-collect.yml
  schedule_cron: "3 */2 * * *"
  timeout_minutes: 5
  secret_names:
    ssh_private_key: SSH_PRIVATE_KEY
    host: FORGE_HOST
    user: FORGE_USER
  ssh:
    connect_timeout_seconds: 15
    strict_host_key_checking: true
  sources:
    - type: docker
      container: medusa
      severity_profile: high
      lines: 5000
      since: 2h10m
    - type: file
      path: /var/log/nginx/error.log
      format: nginx-error
      severity_profile: high
      lines: 5000
    - type: journalctl
      severity_profile: medium
      since: 2h
```

## Log Sources (Level 2 mapping)

```yaml
log_sources:
  - file: logs/docker-medusa.log
    parser: docker                # docker | nginx-access | nginx-error | journald
    source_name: medusa
    severity_profile: high        # high | medium | low
```
